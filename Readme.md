# API de Blog - HeavyApps.Blog

Esta é uma API simples para gerenciamento de posts de blog, desenvolvida em .NET 8.

## Funcionalidades

- CRUD de posts (criar, listar, atualizar, remover)
- Associação de posts a autores
- Controle de status dos posts (Pendente, Publicado, etc.)
- Categorias, resumos e imagens para os posts

## Requisitos

- .NET 8 SDK
- Banco de dados relacional (ex: SQL Server, SQLite)
